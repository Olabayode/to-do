# To-Do App

This project is a React to-do application built to manage tasks with the `useReducer` hook instead of `useState`. It allows users to create, update, delete, and complete tasks while keeping task data saved in `localStorage`.

## Features

- Add new tasks using a controlled form.
- Manage application state with `useReducer`.
- Update existing tasks without changing their position in the list.
- Delete tasks from the task list.
- Mark tasks as complete or incomplete.
- Completed task cards turn green.
- Store tasks in `localStorage` so they remain after refreshing the page.
- Show a readable task date with the month written in words.
- Use `react-icons` for complete, update, undo, and delete buttons.
- Display tasks in a modern card-based layout.

## Components

- `TodoForm`: Handles the task input and add button.
- `TodoList`: Displays the list of tasks.
- `TodoItem`: Displays each task card and its action buttons.
